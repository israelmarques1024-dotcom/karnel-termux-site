Quick Start

After installation

# Install specific AI tools
$ omni install ai --opencode --claude-code --ollama

# Install full environment
$ omni install lang db dev npm shell ui deploy

# Open interactive menu
$ omni

# Check environment health
$ omni doctor

# List available AI tools
$ omni list ai

# Save a memory to Second Brain
$ omni brain save

# Deploy via Vercel
$ vercel deploy --prod

# Deploy via Railway
$ railway up
