Quick Start

After installation

# Install specific AI tools
$ omni install ai --opencode --claude-code --ollama

# Install everything
$ omni install lang db dev npm shell ui

# Open interactive menu
$ omni

# Check environment health
$ omni doctor

# List installed AI tools
$ omni list ai
