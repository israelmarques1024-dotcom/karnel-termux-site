**Omni Catalyst** é um ambiente de desenvolvimento modular para Termux no Android.

Este site contém a documentação oficial da CLI Omni Catalyst.

Criado por **israel marques** (tenho 12 anos).

---

## Tecnologias

- React + Vite + TypeScript
- Tailwind CSS
- wouter (routing)
- Lucide React (ícones)
- Vercel (deploy)

---

## Executar Localmente

```bash
pnpm install
pnpm dev
```
